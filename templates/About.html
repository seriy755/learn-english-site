Данный проект создан в рамках обучения на Цифровой Кафедре МФТИ.
Функциональные воможности проекта:
  1. Словарь слов
  2. Список учебных материалов
  3. Карточки слов
Словарь слов предоставляет пользовтелю возможность самому добавлять, удалять и редактировать слова.
Добавление новых слов при нажатии на кнопку "Добавить новое слово". Откроется страница с формой для добавления нового слова.
Редактирование слов при нажатии на кнопку "ручка". Откроется страница с формой для редактирования выбранного слова.
Удаление слов при нажатии на кнопку "корзина". Выбранное слово удалится из словаря.

Список учебных материалов предоставляет пользовтелю возможность самому добавлять, удалять и редактировать учебные материалы.
Добавление новых учебных материалов при нажатии на кнопку "Добавить учебный материал". Откроется страница с формой для добавления нового учебного материала.
Редактирование учебных материалов при нажатии на кнопку "ручка". Откроется страница с формой для редактирования выбранного учебного материала.
Удаление учебных материалов при нажатии на кнопку "корзина". Выбранный учебный материал удалится из списка.

Карточки слов предоставляют пользователю возможность для изучения новых слов или проверки своих знаний. По щелчку на карточку она перевернётся и покажет перевод слова. Навигация по карточкам осуществляется кнопками "Предыдущая" и "Следующая".
